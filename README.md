# GoDone Mobile

GoDone Mobile adalah aplikasi Flutter untuk membantu pengguna mengelola task, jadwal, prioritas, dan fokus kerja.

## Fitur

- Login dan register akun
- Kelola task, kategori, deadline, dan prioritas
- Kalender task
- Pomodoro timer
- Notifikasi pengingat task
- Profil pengguna dan tema gelap/terang

## Teknologi

- Flutter
- Dart
- REST API

## Cara Menjalankan

```bash
cd frontend
flutter pub get
flutter run
```

```text
frontend/   Aplikasi mobile Flutter
```
